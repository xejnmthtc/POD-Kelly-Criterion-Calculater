# Privacy Policy

This extension does not collect, sell, or share personal data.

## Data the extension stores locally
- User inputs: bankroll and fractional Kelly factor
- Last clicked odds value (to display and calculate in the popup)

All stored data remains on the user’s device using Chrome extension storage.

## Permissions
- storage: used only to save the values listed above
- host permissions (pinnacleoddsdropper.com): used only to read the odds text when the user clicks it

## Remote code
The extension does not download or execute remote code.
